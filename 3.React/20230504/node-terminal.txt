node : node 버전확인 겸 터미널 node로 열음
.exit : node 터미널 나가기
ls : 폴더 내에 있는 구조들 확인
cd 폴더이름 : 그 폴더 안으로 들어감
cd .. : 들어가있는 폴더 상위로 나가기
npx create-react-app 폴더이름 : node-module가 들어있는 react프로젝트 폴더를 만듬
npm start : node-module이 들어있는 폴더에서 실행시켜야함. localhost:3000 으로 라이브서버가 켜짐
ctrl + c : localhost 끄기
npm run build : 파일들을 하나의 번들로 묶어서 빌드함. 그러면 build라는 폴더가 열림
